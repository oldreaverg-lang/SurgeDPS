5. Issues Found & Remediation Status
✓ FIXED — HIGH

Issue 3: med_yr_blt/num_story Never Passed to Damage Calc

estimate_damage_from_raster() called estimate_building_damage() without passing med_yr_blt, num_story, or occtype. The construction era adjustment (+7pp for pre-1970) and multi-story contents reduction were silently never applied.

Fixed: Added 3 missing parameters to the function call.
✓ FIXED — LOW

Issue 7: Inundation Mask Boundary

Buildings where surge exactly equaled foundation height got zero damage. Water at floor level causes moisture and mold damage in reality.

Fixed: Changed threshold from depth_ft <= ffh to depth_ft < ffh - 0.1
OPEN — HIGH

Issue 1: OSM Buildings Have No Replacement Values

When NSI is unavailable, loss = sqft × flat $150/sqft × random ±40% multiplier. The multiplier is seeded by building ID — pure noise, not data. Census ACS median home values by county would be a far better fallback.

Recommended: Query Census ACS API for county-level median home values.
OPEN — HIGH

Issue 2: No Depth Uncertainty Propagation

Surge depth treated as exact truth, but the parametric model has ±13.5% mean error on peak surge. Per-building depth is even noisier. Loss estimates should be confidence intervals, not point estimates.

Recommended: Report [low, best, high] loss using ±30% depth sampling.
OPEN — MEDIUM

Issue 4: Contents-to-Structure Ratio Fixed at 50%

HAZUS default 50% is wrong for many commercial types. Restaurants ~75%, wholesale ~100%, heavy industrial ~150%. NSI val_cont covers most cases but fallback is inaccurate.

Recommended: Use occupancy-specific ratios from HAZUS Table 5.7.
OPEN — MEDIUM

Issue 5: No Wind Damage in Loss Calculation

Frontend shows wind vs. water attribution visually, but actual loss is 100% flood-based. Wind can be 30–60% of total loss for Cat 3+. Critical for coverage determination.

Recommended: Add HAZUS hurricane wind-damage curves; report wind/flood losses separately.
OPEN — MEDIUM

Issue 6: Damage Categories Don't Match FEMA IHP

Our minor/moderate/major/severe thresholds (10/30/50%) are approximations. FEMA IHP uses different eligibility thresholds based on loss amount, not percentage.

Recommended: Add FEMA IHP eligibility estimate based on loss dollar amount.
